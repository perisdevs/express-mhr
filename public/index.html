Sorting

Use URL queries to sort responses

sortBy - Specify an object parameter to sort by

sortOrder - Specify the sorting order ('asc' or 'desc')

Example - https://www.mhriseapi.com/weapons?sortBy=damage&sortOrder=desc

Filters

Results can be filtered for specific object params using URL queries

Example - https://www.mhriseapi.com/weapons?type=insectGlaive&element=Fire
